## Akinator
#  Realization akinator using tree
---
# Install

    $ git clone https://github.com/StaVan28/iLab.git
    $ cd iLab/7_akinator
    $ make install

---
# Usage

Launch
	
	./Bin/akinator

Then choose mods

	1 -- finding items.   Akinator try guess your word.
    2 -- definition mode. Akinator find differences between 2 words.
